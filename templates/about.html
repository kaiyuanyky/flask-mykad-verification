MyKad Verification
Web Service

TPT3101 Project

Project Overview

From the aspect of verifying a MyKad document, it will ease the whole verification process with the web service by using an application.

Project Objectives

The main objective of this project is to design the algorithms for the web service of verifying a MyKad document with different aspects of detections.

Technological Background

In order to perform verification in regards to image processing, it is important to use computer vision as it can be used to recognize the objects on an image.

Project
Overview

This project is to verify the MyKad documents of the users by using a mobile application, with the captured image of MyKad sent by their mobile devices to the web service for processing. In this process, the images are separated into key features that are used to identify and verify the users based on specific criteria. These key features include the detections from aspects of landmarks, security features, and fraud.

Project
Objectives

The main goal of this project is to design the algorithms for the web service of verifying a MyKad document with different aspects of detections. It is to improve the accuracy of verification using this multi-factor verification. Furthemore, this project aims to develop a prototype application to demonstrate the algorithms of MyKad verification web service.

To identify the features for the MyKad images for verification purposes.

To design the algorithms of landmark detection, security feature detection, and fraud detection that can be used in the verification function.

To develop a prototype application to demonstrate the algorithms.

Technological
Background

Python is mainly used for developing the algorithms of different detections.

OpenCV is used for processing the MyKad images.

Tesseract OCR is used for extracting the text from the landmark images.

Flask is used for building this web service along with HTML, CSS, and JS.
